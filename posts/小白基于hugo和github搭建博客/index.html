小白基于Hugo和Github搭建博客

Dec 21, 2023
1 min read

前言

本文是一个纯小白搭建博客的教程。是在win11系统内进行操作。

主要参考《使用hugo+github搭建免费个人博客》、《利用SSH将本地仓库文件推送至远程仓库（GitHub）》和网络上的部分文章。

一、下载git

二、下载Hugo

1、下载

点击此进入Hugo下载界面。

依次点击Code、Download ZIP下载安装包

在任意盘符下新建Hugo\bin文件夹

例如我建立如下文件夹C:\Program Files\Hugo\bin

把下载好的hugo安装包解压到此文件夹下

此时bin文件夹下应该有如下文件

hugo.exe
LICENSE
README.md

在bin下右键选择Open Git Bash here打开Git命令窗口

输入

    hugo version

出现Hugo版本号即为成功

2、配置环境变量

右键此电脑依次点击属性、高级系统设置、高级、环境变量

在系统变量中双击path进入后新建文件路径。路径为bin文件夹所在的位置

例如我的路径为C:\Program Files\Hugo\bin

加入成功后搜索cmd打开终端输入

hugo version

出现Hugo版本号即为成功

三、本地搭建博客

1、搭建站点

在任意文件夹下右键打开Open Git Bash here输入

hugo new site test_blogs

其中test_blogs可以换成名字

ps.我就选择在C:\Program Files\Hugo下运行此命令

运行成功后就会在C:\Program Files\Hugo下看到一个名为test_blogs的文件夹

进入文件后会有如下文件

archetypes
assets
content
data
i18n
layouts
static
themes
hugo.toml

此时站点就搭建成功了

2、下载主题

点击此进入Hugo博客主题

选择一个喜欢的主题下载，这里我以m10c为例，下载地址为：https://github.com/vaga/hugo-theme-m10c

依次点击Code、Download ZIP下载压缩包

把压缩包解压到C:\Program Files\Hugo\test_blogs\themes下

并把主题文件重命名为主题文件

把C:\Users\A\Desktop\blog\themes\m10c\exampleSite里的config.toml文件改成hugo.toml并粘贴覆盖到C:\Program Files\Hugo\test_blogs

把第三行themesDir = "../.."删除

注意

1、一定是要下载安装包而不是以命令安装主题，我搭建博客时就在这一步卡了好久，具体原因不祥

2、命令不要打错，我布置博客时把theme打成themes，搞错了好多遍

3、粘贴覆盖时备份原本文件是一个好习惯

3、新键文章

4、查看本地博客

保存退出后到目录C:\Program Files\Hugo\test_blogs下右键打开Open Git Bash here依次输入

hugo  
hugo server

这时在浏览器主页输入https://localhost:1313就可以看到自己的博客主页了

与此同时目录C:\Program Files\Hugo\test_blogs下也生成了一个文件夹public

四、把博客上传到Github Pages实现线上查看

1、创建github仓库

登录github账户点击右上角的加号，选择New repository

要想博客部署到GitHub上名字这一栏就要有特殊要求

必须为"你的github名字".github.io

例如我的仓库名称为A-WTY.Github.io

选择public、Add a README file

点击Create repositoty创建仓库成功

2、本地仓库和github连接

在C:\Program Files\Hugo\test_blogs\public文件夹下右键打开Open Git Bash here依次输入

ssh-keygen -t rsa -C "你的邮箱号"

一路回车，如果有(y/n)?之类的选择y

在C盘用户文件夹下搜索.ssh文件

搜索出 id_rsa 和 id_rsa.pub 两个文件，它们是 SSH key 的秘钥对。id_rsa 是私钥，不能轻易泄露， id_rsa.pub 是公钥，可以让他人知道。

进入id_rsa.pub文件里

复制里面所有的内容

在GitHub里点击设置

找到SSH and GPG keys里的New SSH key

在Title里随便填key里面粘贴id_rsa.pub文件里的内容

点击Add SSH key加入密钥

在C:\Program Files\Hugo\test_blogs\public文件夹下右键打开Open Git Bash here依次输入

git init 
git add . 
git commit -m "first commit"
git branch -M main
git remote add origin  git@github.com:name/name.git
git pull --rebase origin main
git push -u origin main

其中git@github.com:name/name.git是你Github的名字

first commit是对提交到git缓存区的文件相对上一次修改的的内容提示，这里可以随便填

这时打开Github发现仓库里有本地pbulic文件夹里的文件即为成功

这时在浏览器地址输入"你的github名字".github.io就可以进入博客了
